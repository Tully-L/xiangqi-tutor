# 象棋入门 · Xiangqi Tutor

一页学会中国象棋的教学网站。纯静态、零依赖、单页。

- **棋盘** — 九路十线、楚河汉界、九宫图解
- **七种棋子** — 每张卡片内嵌可交互小棋盘，点棋子看走法，蹩马腿 / 塞象眼 / 炮架一试便知
- **闯关学棋** — 六个循序渐进的关卡：吃子 → 马绕日字 → 隔山打炮 → 将军 → 解将 → 残局擒王，走错有讲解，进度存本地
- **跟教练下一局** — 与 minimax AI 对弈整局，教练实时点评：选子讲规则、失子复盘、威胁预警、提示与悔棋

## 本地运行

```bash
python3 -m http.server 8765
# 打开 http://localhost:8765
```
