# 象棋入门 · Xiangqi Tutor

一页学会中国象棋的教学网站。纯静态、零依赖、单页。

- **棋盘** — 九路十线、楚河汉界、九宫图解
- **七种棋子** — 每张卡片内嵌可交互小棋盘，点棋子看走法，蹩马腿 / 塞象眼 / 炮架一试便知
- **实战演练** — 完整开局对弈，红先黑后，自动轮换

## 本地运行

```bash
python3 -m http.server 8765
# 打开 http://localhost:8765
```

## 测试

规则引擎（`engine.js`）由 TDD 驱动，覆盖七种棋子全部走子规则与边界情况：

```bash
npm test
```
